## 🚀 Our Public Projects

### [NicheBench](https://github.com/HumanFace-Tech/nichebench)
**🎯 AI Model Benchmarking Framework**

A lightweight, extensible CLI framework for benchmarking AI models on framework-specific tasks. Features LLM-as-a-Judge evaluation, multi-turn conversations, and configuration-driven model management.

**Key Features:**
- 🤖 Multi-turn conversations with iterative refinement
- 📊 3-value scoring system (Pass/Partial/Fail)
- ⚡ Parallel execution with progress visualization
- 🛡️ Runaway protection for model behavior
- 📦 Framework packs (Drupal support, more coming)

**Tech Stack:** Python, Poetry, YAML, LiteLLM

---

### [Whisper Recorder UI](https://github.com/HumanFace-Tech/whisper-recorder-ui)
**🎤 Voice Transcription with AI Enhancement**

A streamlined Progressive Web App that combines speech recognition with AI text formatting. Perfect for voice-to-text workflows with intelligent post-processing.

**Key Features:**
- 📱 Mobile-optimized PWA design
- 🔤 High-quality Whisper ASR integration
- 🤖 LLM-powered text refinement
- ☁️ Local and cloud deployment options
- ⚡ Real-time processing

**Tech Stack:** JavaScript, Web Audio API, PWA, Whisper, LLM APIs
